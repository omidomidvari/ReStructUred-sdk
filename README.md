# restructured sdk

***

### introduction
restructured sdk is a way to run structjs in a terminal using its own format called `struct2` and normal structjs would be
using `struct1

***
> [!WARNING]
> this is still under development so please test it in a vm before acually using it
> **ℹ️ Info:**  
> build it using `tsc` in the root of the repo

***
requierments:
- have nodejs installed with npm
- have tsc installed using `npm install -g typescript`
- have the repo in the current dir
- be in the repo
- have type-nodes installed (optional)

***
run the build (recommended)
```
npm run build
```
(not recommended) run the node script (in src dir with copied pacakge.json)
```
npm run start-nodets
```
2. run it
```
npm run start
```
thats all for the instructions!
